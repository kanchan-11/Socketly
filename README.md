# Socketly 🚀

**Socketly** is a real-time, room-based chat application built with **React**, **Spring Boot**, **WebSockets**, and **MongoDB**.  
It allows users to create or join chat rooms, view previous messages, and chat live in real-time.

---

## ✨ Features (Current)

- Create or join a chat room using a room ID
- Load previous chat messages when joining a room
- Real-time text messaging using WebSockets

---

## 🛠 Tech Stack

**Frontend**
- React.js
- WebSocket (STOMP / SockJS)
- Tailwind CSS (optional styling)

**Backend**
- Java
- Spring Boot
- Spring WebSocket
- MongoDB

**Infrastructure (Planned)**
- Docker
- GitHub Actions CI/CD
- Cloud hosting (AWS / Azure)
